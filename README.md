Projeto em desenvolvimento para gerenciamento de clientes, serviços e agendamentos.

## Tecnologias

- Python
- SQLite
- Custom Tkinter

## Funcionalidades

- Cadastro de clientes
- Agendamento de horários
- Controle de serviços
- Histórico de atendimentos
